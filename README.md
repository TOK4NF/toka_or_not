# toka_or_not

ready or not mais toka remplace judge

# Comment installer :

1. Télécharger la dernière version (ou la version souhaitée) dans https://github.com/TOK4NF/toka_or_not/releases/tag/readyornot

2. prendre le dossier compressé "VO_MOD" et l'extraire

4. mettre le dossier "VO_MOD" dans le dossier content de ReadyOrNot, exemple : 

C:\Program Files (x86)\Steam\steamapps\common\Ready Or Not\ReadyOrNot\Content
